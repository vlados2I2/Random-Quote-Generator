# Описание программы
**Random Quote Generator** - графическое приложение на Python с использованием `tkinter` для генерации случайных цитат из предопределённого списка. Приложение позволяет добавлять собственные цитаты, просматривать историю генераций, фильтровать историю по автору и теме, а также сохранять и загружать историю в формате JSON.

# Возможности:

- **Генерация случайных цитат** — выбор случайной цитаты из базы по нажатию кнопки
- **Добавление новых цитат** — расширение базы цитат через форму ввода с валидацией
- **Просмотр истории** — отображение всех сгенерированных цитат в виде списка
- **Фильтрация истории** — фильтрация по автору и/или теме через выпадающие списки
- **Сохранение в JSON** — автоматическое сохранение истории в файл `history.json`
- **Загрузка из JSON** — восстановление истории при следующем запуске приложения
- **Валидация ввода** — проверка на пустые строки при добавлении новых цитат

# Инструкция по запуску

## Требования:

- Python 3.10 или выше
- Стандартная библиотека Python (tkinter, random, json, pathlib, dataclasses, unittest)

## Запуск:

1. Откройте терминал или командную строку

2. Перейдите в директорию проекта:

   ```bash
   cd random-quote-generator
3. Запустите приложение:

    python main.py

# Примеры использования

1. Заполните все поля формы:
   - **Текст цитаты:** "Всё, что ни делается, — к лучшему."
   - **Автор:** "Михаил Булгаков"
   - **Тема:** "Философия"
2. Нажмите кнопку **"Добавить цитату"**
3. Появится сообщение "Цитата добавлена!", цитата добавится в базу

### Генерация цитаты:

1. Нажмите кнопку **"Сгенерировать цитату"**
2. В верхней части окна отобразится случайная цитата с автором и темой
3. Цитата автоматически добавится в историю в нижней части окна

### Фильтрация:

- **По автору:** Выберите автора из выпадающего списка в разделе "Фильтрация истории" и нажмите "Применить фильтр"
- **По теме:** Выберите тему из выпадающего списка и нажмите "Применить фильтр"
- **По автору и теме:** Выберите оба значения и нажмите "Применить фильтр"
- **Сброс:** Нажмите "Сбросить" для возврата к полной истории

### Сохранение и загрузка:

- История автоматически сохраняется в файл `history.json` при каждой генерации
- При следующем запуске приложения история загружается автоматически
- Нажмите "Очистить историю" для удаления всех записей
